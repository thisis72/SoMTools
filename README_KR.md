# SoMTools

유니티 툴 및 유틸리티 모음입니다.

## 기능

- **Events**: ScriptableObject 기반 이벤트 시스템.
- **ExFunctions**: 일반적인 유니티 타입들을 위한 확장 함수.
- **Logs**: 커스텀 로깅 유틸리티.
- **StringTools**: 문자열 조작 헬퍼.

## 설치 방법

유니티 패키지 매니저(Unity Package Manager)에서 git URL을 사용하여 이 패키지를 설치할 수 있습니다:

1. 유니티 패키지 매니저를 엽니다 (`Window` > `Package Manager`).
2. 왼쪽 상단의 `+` 버튼을 클릭합니다.
3. `Add package from git URL...`을 선택합니다.
4. 다음 URL을 입력합니다:
   `https://github.com/thisis72/SoMTools.git`

## 요구 사항

- Unity 6000.0 이상.
