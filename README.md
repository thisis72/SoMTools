# SoMTools

A collection of Unity tools and utilities.

## Features

- **Events**: ScriptableObject based event system.
- **ExFunctions**: Extension functions for common Unity types.
- **Logs**: Custom logging utilities.
- **StringTools**: String manipulation helpers.

## Installation

You can install this package via Unity Package Manager using the git URL:

1. Open Unity Package Manager (`Window` > `Package Manager`).
2. Click the `+` button in the top-left corner.
3. Select `Add package from git URL...`.
4. Enter the following URL:
   `https://github.com/thisis72/SoMTools.git`

## Requirements

- Unity 6000.0 or later.
